# 🎵 Music Player Application

## 📌 Project Description
The Music Player Application is a desktop application developed using Core Java and Java Swing. It allows users to select and play WAV audio files with a simple and interactive graphical interface.

---

## 🚀 Features

- 🎵 Choose WAV Song
- ▶ Play Music
- ⏸ Pause Music
- ⏯ Resume Music
- ⏹ Stop Music
- 📊 Progress Bar
- ⏱ Song Timer
- 🔊 Volume Control
- 📂 File Menu
- ℹ About Dialog

---

## 🛠 Technologies Used

- Java
- Java Swing
- Java Sound API
- IntelliJ IDEA
